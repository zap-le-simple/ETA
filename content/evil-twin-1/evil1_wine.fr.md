# Evil Twin via Wine

## Sikarugir Creator

- Installer Sikarugir Creator
- Installer le dernier moteur WS12WineSikarugir10.xx
- Créer un New Blank Wrapper

## Configurer Wrapper
- Ouvrir le wrapper
- Monter les isos des CDs 1 & 2
- Cliquez sur Install Software, Choose Setup Executable puis naviguez sur CD1/Setup.exe
- Installez le jeu puis switcher sur l'autre image disque durant le processus
- Changez l'URL Windows App par "C:\Program Files (x86)\Ubi soft\Evil Twin\Splash.exe"
- Puis test Run. Le Splash menu devrait apparaître.

## Finaliser l'installation
 Avant de pouvoir lancer une partie il reste quelques bricoles : 

- Récupérer et installer la [mise à jour update1](/files/)  afin de lancer le jeu sur Windows XP et ultérieur.
- [Installer le no-CD](/files/) car depuis Windows 10, ni SafeDisc ni SecuROM fonctionnent.
- Installer le plugin pour utiliser une manette moderne
- Installer le mod pour jouer en widescreen

Ces informations sont toutes déjà sur la page [Jouer à Evil Twin sur Windows 10](evil1_windows)
